# Sensor de temperatura e umidade 

<p align='center'>Implementação de um leitor de temperatura e umidade utilizando o sensor AHT10 utilizando pico SDK. </p> 

## Pinos utilizados 

Display LCD 240x320:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SCL  -------- (GPIO 18) |
+-------------------------+
| SDA  -------- (GPIO 19) |
+-------------------------+
| RES  -------- (GPIO 20) |
+-------------------------+
| CS   -------- (GPIO 17) |
+-------------------------+
| DC   -------- (GPIO 16) |
+-------------------------+
````

Sensor de temperatura e umidade AHT10:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SCL  --------  (GPIO 1) |
+-------------------------+
| SDA  --------  (GPIO 0) |
+-------------------------+
````
